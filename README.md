## Installation
### Clone repository
```
git clone https://github.com/KleyberJMH/UnderscoOre.git
```
### Setup your API Key
```
#env.d.ts
COHERE_API_KEY = "{HERE YOUR API KEY}"
COHERE_API_GENERATION_URL = "https://api.cohere.ai/generate"
```

### Install needed dependencies
```
npm install
```

### Run developer server
```
npm run dev
```
